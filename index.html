University Facts




Established in 1824 under the name of University of Manchester Institute of Science and Technology, The University carries a lot of history. With it having 25 nobel prize winner representer and strong research presence, the University is ranked #29 worldwide [1]




History

The University was actually formed as a merger between two existing universities: UMIST and Victoria University of Manchester. The Sackville Street Building first opened in 1902. The engineering facility was able to aid in the first world war supplying hydrophones (A microphone which detects sound waves under water.)   Read more




As Manchester was the place of many significant discoveries such as Rutherford "splitting" the atom [2] and the "Manchester baby" the first computer to run a program[3] and more recently, the isolation of graphene (the first known two-dimensional material)in 2004 which won two nobel prizes. [4] it is no surprise that the University is a recognized and highly regarded institution. The University currently teaches over one thousand courses between the faculties of Biology and Health, Science and Engineering, and Humanities.    Read more

Bash Scripting

Bash is a shell scripting language that executes a wide range of commands. You can use bash as a tool for any task such as extracting information. The user can preform these tasks on the terminal using a combination of existing commands. For example, the command "echo" will display a message of the result of whatever command you asked the computer to perform. [1]


Pipelines (denoted "|") are used to connect two commands together, linking the output of the first command with the input of the second. This makes writing tasks easier thus it is a commonly used property. [2]

Loops in bash are used to run a certain command over and over again until the requested result is satisfied. To do so, you usually start with one of the three commands "while","until","for". while runs the command repeatedly as long as it is true, while until will run the command until the result is true. for, however, does the procedure a little differently; when the computer runs the command it will interpret the list in the order that it was listed at and execute the command.
